# 磁吸爬壁机器人电控总览

## 1.爬壁机器人的硬件部分
### 1.1 主控
主控使用的是汇川的PLC  
型号： EASY系列 H5U
```text
H5U是汇川自主开发的新一代小型PLC产品，支持""EtherCAT""总线通信，具备强大的运
动控制和分布式I/O控制功能，可通过FB/FC功能实现工艺的封装和复用，通过
RS485、CAN、以太网和EtherCAT接口可以实现多层次网络通信。
Easy系列全场景紧凑型中小型控制器全系列共8个机型，满足用户对中小型自动化设
备各种需求，适用于严苛体积、多轴运控、温度控制和通信组网等场景。
```




 制动电阻选型流程图如下

```mermaid
flowchart TD
    A([开始]) --> B["确定往复运动周期 T(s)<br>（根据实际工况确定）"]
    B --> C["确定电机转速 V(rpm)<br>（根据实际工况或由汇川驱动调试软件读取）"]
    C --> D["确定负载与电机惯量比 N<br>（参考“调试手册”调整章节惯量辨识内容）"]
    D --> E["计算制动能量 E(J)<br>E = (N + 1) × J × V² ÷ 182<br>J：电机轴空载转动惯量"]
    E --> F["确定电容吸收能量 Ec(J)<br>查阅“选型手册”，确定电容可吸收的最大制动能量"]
    F --> G{ E > Ec ? }
    
    G -- 否 --> H["不需要使用制动电阻"]
    H --> Z([结束])
    
    G -- 是 --> I["需要使用制动电阻"]
    I --> J["计算制动电阻功率 P<br>P = 2 × (E - Ec)"]
    J --> K["使用外接制动电阻"]
    K --> L["降额70%选用外接制动电阻"]
    L --> Z

```
