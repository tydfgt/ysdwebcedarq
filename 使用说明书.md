# Mizuki 博客框架使用说明书

## 1. 项目介绍

Mizuki 是一个基于 Astro 构建的现代、功能丰富的静态博客框架，采用 Material Design 3 设计风格，具有以下特点：

- 基于 Astro 和 Tailwind CSS 构建
- 支持明暗主题切换
- 响应式设计，适配各种设备
- 丰富的特色页面（动漫、日记、友链、相册等）
- 内置搜索功能
- 支持 Markdown 扩展语法
- 数学公式支持
- 图片优化和画廊集成

## 2. 安装步骤

### 2.1 克隆仓库

```bash
git clone https://github.com/matsuzaka-yuki/Mizuki.git
cd Mizuki
```

### 2.2 安装依赖

```bash
# 安装 pnpm（如果尚未安装）
npm install -g pnpm

# 安装项目依赖
pnpm install
```

### 2.3 启动开发服务器

```bash
pnpm dev
```

开发服务器将在 `http://localhost:4321` 启动。

## 3. 配置说明

### 3.1 主要配置文件

主要配置文件位于 `src/config.ts`，包含以下核心配置：

- **站点信息**：标题、副标题、URL 等
- **语言设置**：默认语言（支持英文、中文、日语等）
- **主题设置**：主题色、壁纸模式等
- **特色页面开关**：控制各功能页面的显示
- **导航栏配置**：顶部导航菜单
- **侧边栏配置**：侧边栏组件布局

### 3.2 语言设置

修改 `src/config.ts` 文件中的 `SITE_LANG` 变量：

```typescript
// 定义站点语言
const SITE_LANG = "zh_CN"; // 语言代码，例如：'en', 'zh_CN', 'ja' 等。
```

### 3.3 站点信息配置

修改 `siteConfig` 对象中的相关字段：

```typescript
export const siteConfig: SiteConfig = {
  title: "Mizuki",
  subtitle: "One demo website",
  siteURL: "https://your-domain.com/", // 请替换为你的站点URL
  // ...
};
```

## 4. 内容管理

### 4.1 创建新文章

使用内置脚本创建新文章：

```bash
pnpm new-post <filename>
```

文章将创建在 `src/content/posts/` 目录下。

### 4.2 文章格式

文章使用 Markdown 格式，支持以下特性：
- 前言（frontmatter）配置
- 代码高亮
- 数学公式
- 表格
- 图片
- 视频嵌入

### 4.3 特色页面内容

- **动漫页面**：配置 `src/content/spec/anime.ts`
- **日记页面**：创建 `src/content/diary/` 目录下的 Markdown 文件
- **友链页面**：配置 `src/content/spec/friends.ts`
- **相册页面**：创建 `src/content/albums/` 目录下的相册文件夹

## 5. 部署方法

### 5.1 构建项目

```bash
pnpm build
```

构建产物将生成在 `dist` 目录。

### 5.2 部署到静态托管平台

- **Vercel**：连接 GitHub 仓库自动部署
- **Netlify**：从 GitHub 直接部署
- **GitHub Pages**：使用内置的 GitHub Actions 工作流
- **Cloudflare Pages**：连接仓库部署

### 5.3 本地预览

```bash
pnpm preview --port <端口号> --host 0.0.0.0
```

## 6. 常见问题

### 6.1 端口访问问题

如果无法通过公网 IP 访问，请检查：
- 服务器防火墙是否开放了对应端口
- 云服务安全组是否添加了入站规则
- 网络设备是否配置了端口转发

### 6.2 内容同步

如果遇到内容同步问题，可设置 `CONTENT_REPO_URL` 环境变量指向内容仓库，或手动创建 `content` 目录。

### 6.3 性能优化

- 启用图片优化：确保 `generateOgImages` 配置正确
- 字体优化：启用字体子集压缩
- 减少未使用的特色页面：关闭不需要的页面开关

## 7. 高级功能

### 7.1 音乐播放器

在 `musicPlayerConfig` 中配置音乐源：

```typescript
export const musicPlayerConfig: MusicPlayerConfig = {
  enable: true,
  mode: "meting",
  id: "14164869977", // 歌单ID
  server: "netease", // 音乐源
  // ...
};
```

### 7.2 评论系统

在 `commentConfig` 中配置 Twikoo 评论系统：

```typescript
export const commentConfig: CommentConfig = {
  enable: true,
  twikoo: {
    envId: "https://twikoo.vercel.app",
    lang: SITE_LANG,
  },
};
```

### 7.3 自定义样式

- 修改 `src/styles/main.css` 添加自定义样式
- 修改 `tailwind.config.cjs` 配置 Tailwind 主题
